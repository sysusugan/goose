> 我很喜欢 goose 这个项目。它把本地 AI agent、CLI、桌面端和 MCP 扩展生态结合得很自然，也非常贴近真实工程场景。所以我在自己的 fork 里把 README 和部分文档翻成了中文，并补了一些中文入口，希望更多中文开发者能更顺畅地上手 goose。

goose 是你的本机 AI agent，能够从头到尾自动化执行复杂的开发任务。它不只是给出代码建议，还可以从零搭建项目、编写并执行代码、调试故障、编排工作流，并与外部 API 交互，而且这些都可以以_自主_方式完成。

无论你是在快速验证一个想法、改进现有代码，还是管理复杂的工程流水线，goose 都可以适配你的工作流，并以高精度执行任务。

goose 以灵活性为核心设计，支持接入任意 LLM，并提供多模型配置来平衡性能与成本；它可以无缝集成 MCP servers，同时提供桌面应用和 CLI 两种使用方式，帮助开发者更快推进工作，把精力放在真正重要的创新上。
